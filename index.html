📌 About the Chrome badge on the icon
Android adds a small Chrome badge to PWA icons installed via Chrome — this is an Android OS behaviour that cannot be removed by the app itself. To get a clean icon without the badge, install via Samsung Internet browser instead of Chrome, or use Safari on iPhone.
🤖 Android — Chrome
1
Make sure this page is hosted at an https:// URL (not a local file)
2
Tap the ⋮ menu (top-right of Chrome)
3
Tap "Add to Home screen" or "Install app"
4
Tap Install — the app icon will appear on your home screen
🍎 iPhone / iPad — Safari
1
Open this page in Safari (must be Safari — Chrome on iOS won't work)
2
Tap the Share button ⬆️ at the bottom of the screen
3
Scroll down and tap "Add to Home Screen" ➕
4
Tap Add in the top-right corner — done!
📱 Samsung Internet
1
Open this page in Samsung Internet
2
Tap the ☰ menu (bottom-right)
3
Tap "Add page to" → "Home screen"
⚠️ Not seeing an install prompt? Make sure you are visiting https://wolffewrought.github.io in Chrome or Samsung Internet. The app must be served over HTTPS to install.
💾 Your data saves automatically using localStorage — entries and settings persist between sessions.
